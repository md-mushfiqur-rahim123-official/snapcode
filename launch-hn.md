Title: Show HN: Snapcode – Turn code into beautiful images from your terminal

Body:

I built a CLI tool that converts any code file into a beautiful, syntax-highlighted image ready for social media, docs, or presentations.

Run it with one command:

npx snapcode app.ts --theme dracula

Features:
- 9 built-in themes (Monokai, Dracula, Nord, GitHub Dark/Light, etc.)
- Window frame with macOS traffic lights
- Line numbers
- Auto-detects 190+ languages
- High-quality 2x export
- Works offline after first run

All rendering happens locally – your code never leaves your machine.

https://github.com/md-mushfiqur-rahim123-official/snapcode

Built with TypeScript, Shiki, and Puppeteer.
